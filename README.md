# 🚧 DistributedMQ (Work in Progress)

This project is currently under active development.

**Current Progress**
- ✅ Core producer implementation
- ✅ Redis Streams integration
- ✅ Topic-based message publishing
- ✅ Dockerized local development environment
- 🚧 Consumer Groups
- 🚧 Acknowledgment (XACK) mechanism
- 🚧 Pending Entry List (XPENDING) handling
- 🚧 Visibility Timeout & Redelivery (XCLAIM)
- 🚧 Dead Letter Queue (DLQ)
- 🚧 Performance benchmarking & documentation

The goal of this project is to build a production-inspired distributed message queue, drawing inspiration from systems such as Amazon SQS while exploring concepts like reliable message delivery, fault tolerance, and scalable event-driven architecture.
